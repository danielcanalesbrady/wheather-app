# App de Clima - Módulo 3

## Descripción

Aplicación web para consultar el pronóstico del tiempo en ciudades de Chile. Refactorizada aplicando metodologías modernas de desarrollo frontend.

## Características

- 10 ciudades chilenas con pronóstico actual
- Diseño responsivo (mobile-first)
- Metodología BEM para organización de estilos
- Arquitectura SASS 7-1
- Bootstrap 4 para componentes y grid system

## Temática

La aplicación se enfoca en ciudades de la región de Valparaíso y zonas aledañas de Chile, mostrando condiciones climáticas reales con iconografía representativa.

## Metodología de Estilos

Se aplicó **BEM (Block Element Modifier)** para nombrar las clases CSS:

- `.weather-app` (Block)
- `.weather-app__content` (Element)
- `.place-card--sunny` (Modifier)
